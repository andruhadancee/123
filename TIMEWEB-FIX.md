# Исправление DNS-записи в Timeweb

## Проблема

В Timeweb указано неправильное имя для A-записи: `.wb-team-cyberclub.ru`

**Должно быть:**
- Имя: `@` (или пустое поле)
- НЕ `.wb-team-cyberclub.ru`

## Решение

### Вариант 1: Исправить существующую запись

1. Найди запись с IP `216.198.79.1`
2. Нажми "Редактировать"
3. В поле "Имя" или "Поддомен":
   - **Удали** `.wb-team-cyberclub.ru`
   - Введи `@` (или оставь поле пустым)
4. Сохрани изменения

### Вариант 2: Удалить и создать заново

1. **Удали** старую A-запись с `.wb-team-cyberclub.ru`
2. **Создай новую** A-запись:
   - Тип: `A`
   - Имя/Поддомен: `@` (или пусто)
   - IP-адрес: `216.198.79.1`
   - TTL: 600 (или по умолчанию)
3. Сохрани

## Важно!

- Имя записи должно быть `@` для корневого домена (wb-team-cyberclub.ru)
- НЕ `.wb-team-cyberclub.ru` - это поддомен, а не корневой домен
- Если поле не принимает `@`, попробуй оставить его пустым

## После исправления

1. Подожừ 5-30 минут
2. Вернись в Vercel Dashboard
3. Нажми "Refresh" рядом с доменом
4. Статус должен измениться на "Valid Configuration"
