# Настройка домена в Vercel

## Текущая конфигурация

**Домен:** `wbteam-cyber-club.com`
**Статус:** Invalid Configuration (нужно настроить DNS)

## DNS-запись для добавления

Добавь эту A-запись у регистратора домена:

- **Тип:** A
- **Имя/Хост:** @ (или оставь пустым, зависит от регистратора)
- **Значение/IP:** 216.198.79.1

## Инструкции для популярных регистраторов

### Reg.ru
1. Зайди в личный кабинет → "Домены"
2. Найди домен `wbteam-cyber-club.com`
3. Нажми "Управление DNS-серверами"
4. Добавь A-запись:
   - Поддомен: @
   - Тип: A
   - IP-адрес: 216.198.79.1
5. Сохрани изменения

### Timeweb
1. Личный кабинет → "Домены и поддомены"
2. Выбери домен → "DNS-записи"
3. Добавь запись:
   - Тип: A
   - Имя: @
   - Значение: 216.198.79.1
4. Сохрани

### GoDaddy
1. My Products → Domains
2. Нажми на домен → DNS
3. Add → A record
4. Host: @
5. Points to: 216.198.79.1
6. Save

## После настройки

1. Подожди 5-60 минут (может занять до 24 часов)
2. Вернись в Vercel Dashboard
3. Нажми кнопку "Refresh" рядом с доменом
4. Статус должен измениться на "Valid Configuration"

## Альтернативный вариант (CNAME)

Если A-запись не работает, можно использовать CNAME:
- Тип: CNAME
- Имя: @
- Значение: cname.vercel-dns.com

Но A-запись предпочтительнее, так как Vercel рекомендует новые записи.
