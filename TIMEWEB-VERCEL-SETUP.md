# Подключение домена Timeweb Cloud к Vercel

## Шаг 1: Настройка DNS в Timeweb Cloud

1. Зайди в личный кабинет [Timeweb Cloud](https://timeweb.cloud/)
2. Перейди в раздел **"Домены"** (или "Домены и поддомены")
3. Найди свой домен `wbteam-cyber-club.com` и нажми на него
4. Выбери вкладку **"DNS-записи"** или **"DNS зона"**

## Шаг 2: Добавь A-запись

В DNS-записях домена нужно добавить новую запись:

### Вариант 1: Через веб-интерфейс Timeweb

1. Нажми кнопку **"Добавить запись"** или **"Создать"**
2. Заполни поля:
   - **Тип записи:** A
   - **Имя/Поддомен:** @ (или оставь пустым - это означает корневой домен)
   - **Значение/IP-адрес:** `216.198.79.1`
   - **TTL:** оставь по умолчанию (обычно 3600)
3. Нажми **"Сохранить"** или **"Создать"**

### Вариант 2: Если поле "Имя" требует ввода

- Если нужно указать имя: введи `@` 
- Если поле оставить пустым: оставь пустым
- Некоторые интерфейсы требуют точку `.` - тогда введи `.`

## Шаг 3: Убедись, что DNS-серверы правильные

В Timeweb может быть два варианта:

### Если используются DNS Timewebnodes (рекомендуется):
- Не нужно ничего менять, просто добавь A-запись

### Если используются сторонние DNS-серверы:
- Переключись на DNS Timewebnodes в настройках домена

## Шаг 4: Проверка в Vercel

1. Подожди **5-30 минут** (время обновления DNS-записей)
2. Зайди в [Vercel Dashboard](https://vercel.com)
3. Открой свой проект
4. Перейди в **Settings** → **Domains**
5. Найди домен `wbteam-cyber-club.com`
6. Нажми кнопку **"Refresh"** рядом с доменом
7. Статус должен измениться с "Invalid Configuration" на **"Valid Configuration"**

## Шаг 5: Проверка работы

После того как статус станет "Valid Configuration":
1. Открой в браузере `http://wbteam-cyber-club.com` (без www)
2. Сайт должен открыться

## Если не работает

### Проверь DNS-записи:
- Убедись, что A-запись с IP `216.198.79.1` добавлена
- Имя записи должно быть `@` или пустым

### Проверь время ожидания:
- DNS-записи могут обновляться до 24 часов (обычно 5-30 минут)
- Используй [DNS checker](https://dnschecker.org/) для проверки, видят ли DNS-серверы вашу запись

### Если нужен www-поддомен:
- Добавь еще одну A-запись:
  - Имя: `www`
  - Тип: A
  - Значение: `216.198.79.1`

## Альтернативный вариант (CNAME для www)

Если нужен поддомен `www.wbteam-cyber-club.com`:
- Создай CNAME-запись:
  - Имя: `www`
  - Тип: CNAME
  - Значение: `cname.vercel-dns.com`

Но для корневого домена используй A-запись с IP `216.198.79.1`.
